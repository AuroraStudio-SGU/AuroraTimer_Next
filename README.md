## Project Setup

Power by Yarn

### Install 初始化

```bash
$ yarn install
```

### Development 开发调试

```bash
$ yarn run dev
```

### Build 生产环境部署

```bash
# For windows
$ yarn run build:win

# For macOS
$ yarn run build:mac

# For Linux
$ yarn run build:linux
```
