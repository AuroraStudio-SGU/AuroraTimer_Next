## Project Setup

Power by Yarn

### Install

```bash
$ yarn install
```

### Development

```bash
$ yarn run dev
```

### Build

```bash
# For windows
$ yarn run build:win

# For macOS
$ yarn run build:mac

# For Linux
$ yarn run build:linux
```
